Definition of terms

Non-personal information

Non-personal information refers to information that cannot be used to identify you in any way.

This site

This site refers to Lärka (https://spraakbanken.gu.se/larka) and LärkaLabb (https://spraakbanken.gu.se/larkalabb)

User

User refers to any person using this web site.

Data saved locally

This site saves non-personal information on the user's device to improve the user experience. This information includes language preference.

Data saved remotely

This site saves answers to exercises on the server. All information collected is anonymous. This site also collects information about the user client used to access the web site, such as browser name, browser version, operating system and screen size in order to improve the user experience across devices.

Data collection purposes

This site collects data for research purposes.
